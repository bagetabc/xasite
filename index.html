Новшества версии 1.21+

  • Крафтер (Crafter): Автоматизированный крафт с помощью редстоуна.
  • Испытательные камеры: Новая структура с врагами и лутом.
  • Бриз (Breeze): Новый моб, атакующий ветряными зарядами.
  • Медный набор: Новые блоки, двери и люки из меди.

Системные требования

XALAUNCHER требует стандартных ресурсов для запуска Minecraft Java Edition:

  • Процессор: Intel Core i3-3210 или AMD A8-7600 (или лучше).
  • ОЗУ: Минимум 4 ГБ (рекомендуется 8 ГБ).
  • Видеокарта: Intel HD Graphics 4000 или AMD Radeon R5 (или лучше).
  • Место на диске: Минимум 2 ГБ для файлов игры.
  • ОС: Windows 10/11 (64-bit).
